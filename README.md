# 🍽️ Web-Project: «Мы не то, чем мы кажемся»

> **Официальный сайт элитного ресторана: симбиоз гастрономии, эстетики и философии.**

---

## 🧐 О проекте
Сайт создан для полного погружения гостя в атмосферу заведения еще до визита. Концепция **«Мы не то, чем мы кажемся»** отражена в визуальном языке: минимализм скрывает богатую палитру вкусов и текстур нашего меню.

**Ключевые задачи сайта:**
*   **Презентация:** Ознакомление с философией шеф-повара и интерьером.
*   **Меню:** Интерактивная карта блюд с акцентом на детали.
*   **Сервис:** Быстрое и удобное бронирование столиков онлайн.

## ✨ Особенности интерфейса (Features)

*   **💎 Premium Design:** Использование премиальной типографики и глубоких цветовых решений.
*   **📅 Booking System:** Проработанный UI-кит для формы бронирования.
*   **📸 Visual Storytelling:** Фото-контент высокого качества, раскрывающий детали кухни.
*   **📱 Mobile First:** Идеальное отображение на смартфонах для гостей «на ходу».
