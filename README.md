# 🌐 Web-Project: «Мы не то, чем мы кажемся»

> **Интерактивное веб-исследование о восприятии, масках и скрытой сути вещей.**

---

## 🧐 О чем этот сайт?
Этот веб-проект исследует границы между внешним образом и внутренним содержанием. Через интерактив и визуальные эффекты пользователь проходит путь от первого (часто обманчивого) впечатления до глубокого понимания идеи.

## ✨ Ключевые особенности (Features)
*   **Интерактивные переходы:** Плавная смена состояний элементов, отражающая смену восприятия.
*   **Адаптивность:** Корректное отображение на всех типах устройств (Desktop / Tablet / Mobile).
*   **Визуальные метафоры:** Использование нестандартной сетки и типографики для передачи концепции.
*   **Dark/Light режимы:** (Если применимо) Две стороны одной истории.
